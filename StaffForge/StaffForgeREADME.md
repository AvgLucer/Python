# 🏢 StaffForge

A beginner-friendly Company Management System built entirely in Python using Object-Oriented Programming (OOP).

PyStaff is a simple command-line application that demonstrates the fundamentals of inheritance, classes, objects, constructors, polymorphism, and list-based data storage. It allows users to manage Managers, HRs, and Employees through an interactive CLI.

This project was created as part of my Python learning journey while practicing real-world OOP concepts.

## 🚀 How It Works

The program allows the user to choose between different operations:

1. Add Manager
2. Add HR
3. Add Employee
4. Display All Records
5. Exit Program

Every new record is stored as an object inside a Python list and can later be displayed.
