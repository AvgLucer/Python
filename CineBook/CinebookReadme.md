# 🎬 CineReserve

A simple command-line Movie Ticket Booking System developed in Python.

---

## 📌 Overview

CineReserve allows users to simulate booking movie tickets through a console interface. Users can choose multiple movies, select seat categories, and receive a final bill based on their selections.

This project was built to practice core Python programming concepts and demonstrate how a simple booking system can be implemented without external libraries.

---

## ✨ Features

- Book multiple movie tickets
- Choose from several movies
- Different ticket prices for each movie
- Select seat categories
- Automatic total price calculation
- Displays booking summary
- Beginner-friendly implementation

---

## 🛠 Technologies Used

- Python 3
- Console/Terminal Interface

---

## 📚 Concepts Demonstrated

- Variables
- Lists
- User Input
- For Loops
- Conditional Statements
- Arithmetic Operations
- Basic Data Storage
- Functions of Python's built-in `sum()`

---

## 🎟 Seat Categories

| Seat Type | Additional Cost |
|-----------|----------------:|
| Basic | $0 |
| Exclusive | +$5 |
| Recliner | +$10 |

---

## 📋 Booking Flow

1. Enter the number of tickets.
2. Select a movie for each ticket.
3. Review selected movies.
4. Choose a seat type for every booking.
5. View the final total amount.
