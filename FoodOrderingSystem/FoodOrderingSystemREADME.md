# 🍽️ Online Food Ordering System (Python CLI)

A command-line based **Online Food Ordering System** built using **Python**. This project simulates the basic workflow of a food delivery application where users can select restaurants, order multiple dishes, calculate the total bill, and choose a payment method.

---

## 📌 Features

* 🍴 Choose from **4 different restaurants**
* 📋 Restaurant-specific menus
* 🛒 Order multiple dishes in a single session
* 💰 Automatic bill calculation
* 💳 Multiple payment methods:

  * UPI
  * Debit Card
  * Credit Card
  * Cash
* ✅ CVV validation for card payments
* ❌ Handles invalid restaurant and menu selections
* 🔄 Loop-based ordering system

---

## 🏪 Restaurants Included

* Jaba Ka Dhaba
* Kanpur Fried Chinese
* Surya Credit Restaurant and Bar
* La Bella Baguette

Each restaurant contains its own unique menu and pricing.

```bash

```

2. Navigate to the project folder.

```bash

```

3. Run the program.

